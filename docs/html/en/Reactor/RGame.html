Dispose Method

Releases all resource used by the Reactor.RGame object. This is where you would also remove any user loaded content.

Syntax

public abstract void Dispose ()

Remarks

Call RGame.Dispose when you are finished using the Reactor.RGame. The RGame.Dispose method leaves the Reactor.RGame in an unusable state. After calling RGame.Dispose, you must release all references to the Reactor.RGame so the garbage collector can reclaim the memory that the Reactor.RGame was occupying.

Render Method

Override for control over rendering. This is always required. This will be called everytime a frame needs to be rendered.

Syntax

public abstract void Render ()

Remarks

Documentation for this section has not yet been entered.

Example

Example render loop. Clear must be called first, render anything you wish, then call Present() to swap the back buffer and show it on screen.
Example
            public override void Render()
            {
            Engine.Clear();
            // TODO: Render your game objects or call Scene.RenderAll(); if you wish for the scene manager to attempt to render them all.
            Engine.Present();
            }

A typical update loop. Anything your game needs to update should be called within this method. If an object has an Update() method, it needs to be added somewhere within this method block flow.
